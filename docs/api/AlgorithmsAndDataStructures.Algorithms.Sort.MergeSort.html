MergeSort_Recursively_Wrapper(List<Element>)

This is to be able to call MergeSort sort methods with only the list that needs to be sorted, and independent of the indexes. This is needed for methods that receive other sort methods as parameters, and would ideally like to have similar signature for all the methods that are passed as parameters, In sort methods the signature is: void SortMethod(List{int} list);
