RotateLeft(TNode, TNode, Int32)

Rotates a key from the right sibling of the node via their parent to the node. The cost of this operation is at inserting keys and children, in right position (to preserve order), Which at worst is O(K), Where K is the maximum number of keys in a node, and thus is constant.
