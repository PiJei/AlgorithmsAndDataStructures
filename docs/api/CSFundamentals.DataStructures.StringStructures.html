Namespace CSFundamentals.DataStructures.StringStructures

Classes

LLPPS

Implements the length of the longest proper prefix of a string which is also a suffix.

StringSuffix

Stores information about a suffix of a string.

SuffixArray

Implements SuffixArray data structure. A suffix array of an string is an array of integers that contains the starting index of all suffixes of the string in alphabetically sorted order. For example for string 'data' , where suffixes are : 'data', 'ata', 'ta', 'a' the suffix array is [3, 1, 0, 2]

SuffixTree

Implements a SuffixTree also known as a PAT. A DFS search of the tree should give the collection of all the suffixes of the string. This implementation is Naive: not optimized.

SuffixTreeNode

Implements a suffix tree node.
