Search(String, String)

Implements Knuth-Morris-Pratt algorithm to search for all the occurrences of pattern in text. Uses proper prefixes, and requires preprocessing of pattern. The idea while searching for pattern in text, is that the algorithm has already 'seen' some characters in text, so it should not check again whether those characters match substrings of pattern. Compared to NaiveSearch, where at each internal iteration, j (iterator over pattern) is reset to zero, here j is not reset to zero always. The value j is set to, depends on its prefixes.
